# Research

This directory is intended for experiments, research, and validation code related to the underlying frameworks used by the project.

Target underlying frameworks include:
- `wgpu`
- `winit`
- `vello`
- `lyon`
- `cosmic-text`
- `taffy`
- `tao`
- `glyphon`
- `masonry`

Suggested structure:
- `notes/`: research notes and conclusions
- `prototypes/`: experiment code for validating framework features
- `benchmarks/`: performance tests and comparison data

Continue to add subdirectories and documentation as the research evolves.
