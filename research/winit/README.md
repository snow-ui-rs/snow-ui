# Winit Research Examples

This directory contains research examples for `winit` and is organized for multiple sample programs.

Structure:
- `examples/`: multiple example binaries for different winit experiments
- `src/`: shared code and helper utilities used by examples

Run the minimal example:

```bash
cd research/winit
cargo run --example minimal
```

The minimal example creates a window and exits when the window is closed or the Escape key is pressed.
